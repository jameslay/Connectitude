Instructions

Search - Click the "Search" button to start the home automation, information will be gathered every 30 second. Timer will be reset everytime the "Search" button is clicked.
Information for each room for all houses will be gathered and displayed if no house name is entered. If a house ID is entered, all rooms for that house will be gathered and displayed. All houses and their IDs can be found under "Houses".
The ID for each house is displayed within the () after the house name.

Clear - Click the "Clear" button to clear all rooms, input and stops the timer.

Add house - Click the "Add house" button to add another house.

Add room - Click the "Add room" button to add a room to a specific house.

Adding/deleting rooms and deleting houses will display all rooms afterwards if the action was successful!
